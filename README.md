# AI Comic Generator (AI 漫畫產生器)

這是一個基於 Google Gemini API 的 AI 漫畫生成工具。它結合了兩個強大的介面：一個直覺的 **Web 應用程式 (Web App)** 用於規劃故事與分鏡，以及一個 **Antigravity Skill** 用於直接在對話中進行多回合 (Multi-Turn) 的漫畫生成與圖片後製編輯。

🔗 **線上版本**: [https://Wolke.github.io/ai-comic-generator/](https://Wolke.github.io/ai-comic-generator/)

---

## 1. Web App: 故事規劃與分鏡工作室

Web App 的主要目標是幫助你**將文字故事轉換為結構化的漫畫分鏡**，並管理角色與場景設定。

### 基本流程

1. **設定 API Key**
   - 點擊右上角的「設定」圖示 (`Settings2`)。
   - 輸入你的 Gemini API Key（可選擇儲存在瀏覽器中）。
   - 選擇你想使用的腳本生成模型（例如 `gemini-2.5-flash`）與圖片生成模型（推薦 `gemini-3.1-flash-image-preview`）。

2. **輸入故事與生成分鏡**
   - 回到「工作室」頁面。
   - 在左側的「故事輸入」區塊寫下你的故事。
   - 點擊「生成分鏡腳本」。AI 會自動分析故事，提取出【角色】、【場景】，並切分出【漫畫頁面】與【分格 (Panels)】。

3. **管理角色與場景 (Reference Images)**
   - 在「角色設定」與「場景設定」區塊，你可以看到 AI 提取出的名單。
   - 你可以點擊 **「AI 生成」** 讓模型幫你畫出設定圖，或是點擊 **「上傳」** 使用你自己的圖片作為角色/場景的參考圖 (Reference Image)。
   - **參考文字描述開關**：當你上傳了參考圖，下方會出現一個 toggle 開關。你可以決定 AI 在生成此角色/場景時，是否還要參考文字描述（關閉可以避免文字描述與你的上傳圖片衝突）。
   - 點擊描述文字可以直接編輯角色或場景的設定。

4. **生成漫畫頁面**
   - 展開右側的「頁面」區塊。
   - 每頁都有一個 **「顯示 Prompt」** 區塊，這是一個文字輸入框，包含了 AI 將送出給繪圖模型的完整分鏡指令（包含鏡頭、動作、角色對應、對話等）。
   - **你可以自由修改這個 Prompt**（例如把「特寫」改成「遠景」，或修改對話）。如果改壞了，可以點右上角的「重設為自動生成」。
   - 點擊「生成此頁」，AI 就會根據 Prompt 和你設定好的角色/場景參考圖，畫出一整頁的漫畫。

5. **預覽與分享**
   - 生成完成後，點擊圖片即可全螢幕預覽。
   - 點擊每頁右上角的分享按鈕，可以將圖片下載、複製到剪貼簿，或是分享到社群媒體。

## 2. Antigravity Skill: 對話式漫畫生成與多輪編輯 (Multi-Turn Editing)

雖然 Web App 很適合用來結構化地規劃長篇故事，但如果你想要**針對單一圖片進行來回修改（例如：「把第一格的貓放大」、「讓主角手裡拿杯咖啡」）**，目前最新的 Gemini Image API 支援一種強大的「對話式修圖 (Multi-turn Editing)」功能。

我們將這個功能做成了 Antigravity Skill，讓你可以直接在對話框裡完成這些魔法。

### 如何使用 Skill

在 Antigravity 的對話框中，你可以直接下達指令。Skill 會強制執行**兩步確認流程 (2-Step Workflow)**，確保生成結果符合你的預期：

#### Step 1: 請求生成 & 審核分鏡腳本
直接告訴 Antigravity 你想畫什麼。如果專案裡已有 Web App 生成的資料，你可以請它參考。

> **你**：「幫我畫一頁 4 格漫畫，角色是一隻橘貓在公園散步，寫實風格。」

Antigravity **不會馬上生圖**。它會先寫出一份 Markdown 格式的「分鏡腳本草稿 (Storyboard Draft)」，並要求你確認：

> **Antigravity**：「這是我規劃的四格分鏡腳本，請確認：...（顯示草稿）... 請問有哪裡需要修改嗎？」

你可以回覆：「第三格改成仰角」、「第四格加一句台詞 'Zzz'」，直到你滿意為止，最後說「OK，就照這樣畫」。

#### Step 2: 圖片生成與參考圖注入
你確認腳本後，Antigravity 就會呼叫繪圖模型。此時你可以要求它加入參考圖：

> **你**：「OK，請開始畫。對了，橘貓請參考我上傳的這張照片。」（並附上一張貓的圖片）

Antigravity 會將你的腳本與參考圖結合，生成出完美的漫畫頁面，並顯示在對話中。

#### Step 3: 多輪編輯 (Follow-up Edits)
這是 Skill 最強大的地方。如果生成的圖片有需要微調，**不要重新生成一整張**，直接告訴它你想怎麼改：

- **新增物件**：「在每一格女主角出現的地方，讓她手裡拿著一杯咖啡。」
- **改變場景**：「把這四格的背景，全部換成我新上傳的這張室內泳池照片。」
- **調整畫面**：「把第二格的蝴蝶變成一隻小鳥。」

Antigravity 會把之前的圖片當作「底圖」送回給 API，並套用你的修改指令，這就是為什麼角色外觀和分鏡排版都不會跑掉的原因！

## 開發與部署

這個專案使用 React 19 + Vite 6 + Tailwind CSS 建構。

```bash
# 安裝依賴
npm install

# 本地端開發 (服務於 http://localhost:3000/)
npm run dev

# 建置 Production 版本
npm run build

# 部署到 GitHub Pages
npm run deploy
```

> **注意**：本地開發時，Vite 會將 base path 設為 `/`。只有在執行 `build` 準備部署時，base path 才會切換為 `/ai-comic-generator/` 以符合 GitHub Pages 的路徑規範。
